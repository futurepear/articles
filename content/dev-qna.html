All Questions and Answers
  • Q:Why if we go to level 60 then die we go back to level 35?
  • A:When you die you lose some of your levels, and the maximum level you can spawn at is level 35.
  • Q:zombie boss???
  • A:Thats Wormy from the Halloween gamemode. It may be added to FFA soon.
  • Q:all of the bosses are too easy can you make a harder one. Just an idea but maybe if you challenge the castaway and defeat him/her/it you can reset the boss timer. can you make castaway like the final boss?
  • A:We may add more difficult bosses or more difficult optional versions of current bosses, but there are no garuntees.
  • Q:Can you add a team battles?
  • A:Gamemodes will be added when the playerbase increases. There is no set date for this.
  • Q:Can you nerf the "small" upgrade? The speed bonus that comes with it is unnecessary, because being a smaller target is already a huge buff.
  • A:We will do a poll for this and decide based on that.
  • Q:Do you have plans to add an adventure mode ? (multiplayer/singleplayer)
  • A: We have plans for something similar. The details aren't decided yet.
  • Q:can you make ally button somewhere so we dont hurt each other when we in a ally
  • A: No. We do not want to support teaming in the FFA gamemode.
  • Q:Can you add the update where we can add and create our own servers and private rooms can friends can get invited to with the invite link? As the current 'add server' is confusing.
  • A: This is a planned feature for the future. Right now the add server button can add a link to an existing server, but the only existing server is the main server.
  • Q:Either way, do you have plans to add a interactive tutorial (stoppable) that tells new players how to actually play the game, as new players may be confused of the current tutorial.
  • A: Yes, we will most likely add it with the client rewrite.
  • Q:What plans do you have to add to the game?
  • A: We are currently working on rewriting the client and creating a new rendering engine for it. It will help reduce lag, make the experience smoother, and will make the game look much better. The client rewrite will come in a V3 update and will have a bunch of other stuff.
