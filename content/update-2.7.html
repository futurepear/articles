Update 2.7 has arrived!

New Maps + Portals

The old map has been reintroduced to the game! The server now consists of 2 separate maps that are connected with portals. Portals can be found around the map and can be used to access the other map. All bosses have been moved to the 2nd map.

Exodus Map (main)
Bosses Map


UI Changes

The class selector has been changed to an easier-to-understand version. You can now hold down the [TAB] key to see all the classes you unlocked. Progressing through the class tree unlocks classes.



New Weapon: The Pulse Bow

Archer -> Marksman -> Winged Bow -> Pulse Bow

Click once to charge up the arrow, and click again to fire!



Slimes gained sentience

When hitting slimes, slimes will now attempt to run away from the player for a short period of time. Slimes also now blink.



Boss Cooldowns

To prevent boss farming, we added a cooldown between fighting bosses:

Cooldowns:
  • Goblin Walock - 20 minutes
  • Erthium Eye - 30 minutes
  • Mechanical Terror - 30 minutes

Other Changes

  • Made block borders darker
  • Small physics changes
  • Small rendering optimizations
